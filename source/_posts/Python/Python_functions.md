## 数学函数

```py
float abs(float)
float/int max(a1,a2,a3,...) # 这里的参数是一个 sequence，类似于 range() 函数的返回值，和 C 语言不同，很有意思
```

```py
import math
math.sqrt(float)
```

## Python编程常用函数

```py
range(start, stop[, step])
bool isinstance(variable, tuple of type)
```
