## 回显

不回显：使用分号 `;`  
回显：使用回车或逗号 `,`

## 分数模式

```matlab
format rat
```

回到小数模式

```matlab
format
```

## 小数位数更多

```matlab
format long
```

回到短模式

```matlab
format short
```

## 清屏

```matlab
clc
```

## 清变量

```matlab
clear
clear A B
```
