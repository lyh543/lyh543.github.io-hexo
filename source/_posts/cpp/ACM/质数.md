## 求质数

### 埃氏筛法

复杂度`O(nloglogn)`

### 欧拉筛（线性筛）

```c++
bool vis[maxn] = { 0 };
int prime[maxn], cnt = 0;
void euler(int n)
{
	for (int i = 2; i < n; i++)
	{
		if (!vis[i]) prime[cnt++] = i, vis[i] = 1;
		for (int j = 0; j < cnt && i * prime[j] < n; j++)
		{
			vis[i*prime[j]] = 1;
			if (i%prime[j] == 0)
				break;
		}
	}
	for (int i = 0; i < cnt; i++)
		cout << prime[i] << ' ';
}
```
